Polynomial Constant Term Finder

Computes the exact constant term c of a polynomial using Lagrange interpolation with BigInteger fractions for arbitrary large numbers and bases.

Usage

Place Main.java and data.json in the same folder.

Compile:

javac Main.java

Run:

java Main

Output:

Exact constant term (c): 3
Notes

No external libraries or JARs required

Handles extremely large numbers without precision errors
